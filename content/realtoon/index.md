# Realtoonの設定について

## Smooth Object Normal設定方法

- Directional Lightの**Rotationを０**にする
- キャラのRotationを**x: 0, y: 180, z: 0**にする
- Material > Shadow > Show normal: ONにする
- Smooth normal: x/yを調整する
- キャラのRotationを**x: 0, y: 90 z: 0**にする
- Smooth normal: xを調整する

## 影色

- 影色を調整する際は環境光を０にする
- 肌の色と同系色（乗算）
- 紫系統・赤みを薄めに乗せる（20～30%）
